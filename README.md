# Validador, Generador y Gestor de Contraseñas

Una aplicación web que permite generar contraseñas seguras, validar su fortaleza en tiempo real y gestionar una lista de contraseñas guardadas por sitio y usuario.

## Descripción

Este proyecto fue desarrollado como herramienta para crear y administrar contraseñas de forma segura directamente desde el navegador. Cuenta con validación en tiempo real, generación aleatoria con opciones personalizables y una lista donde se pueden ver, editar y eliminar las contraseñas guardadas.

---

## Funcionalidades

- Validación en tiempo real de la contraseña maestra con barra de fortaleza
- Generación de contraseñas aleatorias con longitud y charset personalizables
- Opciones de caracteres: minúsculas, mayúsculas, números y caracteres especiales
- Copiar contraseña al portapapeles con un clic
- Mostrar u ocultar contraseñas con el botón del ojo
- Guardar contraseñas con sitio y usuario (generada o escrita manualmente)
- Editar y eliminar contraseñas guardadas
- Interfaz con dos pestañas: Gestionar y Lista de contraseñas
- Fondo animado con gradiente
